# JungleChess - Dou Shou Qi

This is a TypeScript React app for Dou Shou Qi (Jungle Chess).

## Features
- Play Dou Shou Qi (Jungle Chess) in your browser
- Save and load game state
- Restart game
- Modern UI with React Context for state management

## Getting Started

### `npm start`
Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### `npm test`
Runs tests using Jest and React Testing Library.

### `npm run build`
Builds the app for production to the `build` folder.

## Project Structure
- `src/components/` - UI components
- `src/game/` - Game state and context
- `src/pieces/` - Piece movement and capture logic
- `src/utils/` - Utility functions (save/load)
